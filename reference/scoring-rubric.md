# Audit scoring rubric

Use numeric scores only when coverage and evidence are strong enough.

## Base rubric

```text
0-2  Control absent or systemic risk
3-4  Sporadic, fragile, or unverified controls
5-6  Working baseline with significant gaps
7-8  Good and mostly verified practice
9    Mature, measured, repeatable process
10   Demonstrated excellence, automated and continuously validated
```

## Required disclosures

For each scored category include:

- satisfied criteria
- missing criteria
- penalties applied
- coverage mode
- confidence
- any `N/A`
- denominator used in the overall score

Use `Not scored` if coverage is too weak for a reliable number.
