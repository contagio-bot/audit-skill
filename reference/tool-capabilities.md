# Bundled tool capabilities

This package ships local stdlib utilities under `scripts/`:

- `inventory.py`
- `detect_capabilities.py`
- `normalize_findings.py`
- `compare_findings.py`
- `validate_report.py`
- `export_sarif.py`
- `build_fix_plan.py`

They are helpers for deterministic evidence gathering and normalization.
They do not install external packages or fetch third-party code.
