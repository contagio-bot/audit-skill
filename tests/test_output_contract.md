# Output contract scenarios

## Specialist methodology output

- expected: executive summary, scope/coverage, capability manifest,
  method/evidence, detailed findings, remediation, persistence disclosure

## Structured continuity run

- when baseline/diff/verify/fix-plan is requested, findings should be
  representable using `reference/finding-schema.md`

## Area-specific addendum

- expected: methodology-specific matrix/map/inventory appears in addition
  to the common output sections, not instead of them
